An interactive mathematical brainteaser written in vanilla JavaScript, HTML, and CSS.

  • Dynamically tracks and displays current points and lines based on mouse movement

Whose Democracy

Whose Democracy is a full-stack web app for keeping up with congressional goings-ons. It is roughly based on ProPublica's Represent, but with the added ability to "follow" legislators which form the basis of a customized news feed for each user.

Whose Democracy uses a Postgres database, Ruby on Rails on the backend, and JavaScript with React/Redux on the frontend. It uses SmartyStreets Geocoding API to look up users' representatives based on their street address, the ProPublica Congress API to display up-to-date congressional data, and the OpenSecrets API to display each legislator's top donors.
